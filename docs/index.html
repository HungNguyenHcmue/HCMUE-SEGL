Abstract

Group-based learning has become a cornerstone of modern education, fostering collaboration, critical thinking, and enhanced problem-solving skills among students. In this study, a deep learning framework, called MoGL-Net, is proposed. This framework integrates skeleton-based posture, sequential action features, facial emotion, and classroom context to assess student engagement. Besides, a fusion strategy employing Self-Attention and Cross-Attention mechanisms is utilized to capture interdependencies between them. Additionally, this study also introduces a real-world dataset, HCMUE-SEGL, consisting of 1,663 video segments collected from 61 students participating in small-group learning activities. Experiment results show that the MoGL-Net model achieves an accuracy of 90.70% on HCMUE-SEGL, highlighting the potential of multimodal learning for developing intelligent systems to monitor and provide objective classroom analytics in group learning environments.

Dataset: HCMUE-SEGL

HCMUE-SEGL is a real-world multimodal dataset for student engagement assessment in small-group learning activities. The dataset contains 1,663 annotated video segments collected from 61 students.

  • Task: Student engagement recognition in group learning
  • Number of samples: 1,663 video segments
  • Participants: 61 students
  • Engagement levels: Disengaged, Normal, Engaged, Very Engaged
  • Modalities: skeleton pose, full-body RGB action, facial emotion, classroom context

Method: MoGLE-Net

MoGLE-Net contains four branches: a pose skeleton branch, an RGB action branch, a facial emotion branch, and a group context branch. Behavior and emotion features are first fused using Self-Attention. Then, Cross-Attention is used to align individual behavior-emotion features with group-level classroom context.

Results

  • Accuracy: 90.70%
  • F1-score: 90.44%
  • Precision: 90.83%
  • Recall: 90.70%
